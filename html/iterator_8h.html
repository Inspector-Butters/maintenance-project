Detailed Description

Iterator for a linked list.

Author
Emil Engelin and Erdem Garip
Date
3 Oct 2021

Implementation of an iterator for a linked list. You must destroy the iterator manually (using ioopm_iterator_destroy) to free the memory it uses. The iterator does not take ownership of any values passed back from the underlying list.
