Detailed Description

Simple hash table that maps keys to values.

Author
Emil Engelin and Erdem Garip
Date
3 Oct 2021

A standard hash table implementation of elem-to-elem key-value pairs. You yourself need to free the data passed to the hash table (ie. the hash table doesn't own the data passed to it), but also the hash table itself (using ioopm_hash_table_destroy).
