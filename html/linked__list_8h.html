Detailed Description

Simple linked list that contains values.

Author
Emil Engelin and Erdem Garip
Date
3 Oct 2021

A standard linked list implementation that contains elem values. You yourself must free the memory allocated to the linked list using ioopm_linked_list_destroy, however, the linked list does not own the values given added to it.
