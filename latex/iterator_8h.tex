\doxysubsection{Detailed Description}
Iterator for a linked list. 

\begin{DoxyAuthor}{Author}
Emil Engelin and Erdem Garip 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
3 Oct 2021 
\end{DoxyDate}
Implementation of an iterator for a linked list. You must destroy the iterator manually (using ioopm\+\_\+iterator\+\_\+destroy) to free the memory it uses. The iterator does not take ownership of any values passed back from the underlying list.
