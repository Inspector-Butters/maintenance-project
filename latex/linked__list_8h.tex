\doxysubsection{Detailed Description}
Simple linked list that contains values. 

\begin{DoxyAuthor}{Author}
Emil Engelin and Erdem Garip 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
3 Oct 2021 
\end{DoxyDate}
A standard linked list implementation that contains elem values. You yourself must free the memory allocated to the linked list using ioopm\+\_\+linked\+\_\+list\+\_\+destroy, however, the linked list does not own the values given added to it.
