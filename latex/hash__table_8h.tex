\doxysubsection{Detailed Description}
Simple hash table that maps keys to values. 

\begin{DoxyAuthor}{Author}
Emil Engelin and Erdem Garip 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
3 Oct 2021 
\end{DoxyDate}
A standard hash table implementation of elem-\/to-\/elem key-\/value pairs. You yourself need to free the data passed to the hash table (ie. the hash table doesn\textquotesingle{}t own the data passed to it), but also the hash table itself (using ioopm\+\_\+hash\+\_\+table\+\_\+destroy).
